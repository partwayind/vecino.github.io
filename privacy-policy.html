4. Data Security

We implement robust security measures to protect your information from unauthorized access, alteration, disclosure, or destruction:

4.1 Encryption

  • In Transit: All data transmitted between your device and our servers is encrypted using TLS/SSL protocol
  • At Rest: Sensitive personal information is encrypted using AES-256-GCM encryption
  • End-to-End Encrypted Communications: Chat messages between users are encrypted client-side

4.2 Key Management

Private encryption keys are stored securely in:

  • iOS: Apple Keychain
  • Android: Android KeyStore

4.3 Payment Security

Important: Vecino does NOT store credit card information. All payment processing is handled by Stripe, a PCI DSS Level 1 compliant payment processor. We only receive transaction confirmations.

4.4 Access Controls

  • Row-level security policies enforce database access controls
  • Employee access to user data is restricted to necessary job functions only
  • All data access is logged and monitored

4.5 Limitations

While we implement comprehensive security measures, no system is completely secure. If you believe your account has been compromised, contact us immediately.
